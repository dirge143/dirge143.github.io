Navicat16安装与激活

发表于2024-06-08|更新于2024-06-08
|阅读量:

激活

获取软件

下载激活需要的注册机和Navicat16 链接：链接：https://pan.baidu.com/s/1PC6FYyLKYnzz381jKgxB4w?pwd=2024
提取码：2024

Navicat16安装

下载好文件后，双击安装包，无脑下一步即可安装完成，必要的话可以自行修改安装位置

使用注册机

**注：此注册机建议破解提供的Navicat版本，其他版本或许失败

同时打开Navicat16和 Navicat注册机，在注册机中选择好Navicat的路径点击patch且勾选RSA Key，弹出提示后点击yes（是），注：使用管理员身份打开注册机

点击generate生成许可证

转到Navicat16弹出的试用提醒，点击注册。将生成的激活许可证复制到Navicat中，点击激活（注意断开网络连接）

弹出该窗口时点击手动激活

  1. 点击手动激活之后，只会出现激活码，将激活码复制到 Request Code 框中
  2. 点击 Generate Activation Code，会在 Activation Code 框中生成激活码
  3. 将 Activation Code 框中的激活码复制到左边的激活码框中，点击激活

不出意外则会激活成功显示获得永久许可证
